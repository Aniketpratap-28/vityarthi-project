# vityarthi-project# Personal Health Logger

## Overview
A simple command-line Personal Health Logger to record daily health metrics (water, sleep, exercise, mood) and view basic summaries. Suitable for students and quick prototyping.

## Features
- Add a daily health entry
- View all entries
- View summary statistics (average water, sleep, exercise)
- Simple CSV persistence (health_data.csv)

## Technologies
- Python (standard library only)
